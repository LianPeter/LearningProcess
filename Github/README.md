## 项目选择标准 🎯

我挑选项目的标准很简单：

1. **够实用**：不是花里胡哨的玩具项目
2. **有维护**：持续更新，社区活跃
3. **文档全**：README详细，上手容易
4. **代码优**：值得学习的代码风格

## 如何使用这个仓库？🤓

1. **按需查找**：根据目录找到你感兴趣的部分
2. **深入学习**：每个项目我都会简单介绍为什么推荐
3. **实践运用**：大胆尝试，有问题就提issue
4. **分享交流**：欢迎推荐优质项目！
